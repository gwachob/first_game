# first_game
First C++ ncurses game

## Build
```shell
mkdir build
cd build
cmake ..
make
```

## Run
`build/first_game`

## Playing
Use the arrow keys to move around.

Press the space bar to drop a star.

Press 's' to drop a rotating square.

Press '+'/'-' to speed up/slow down the "world clock" relative to the game ticks.

Press 'x' to quit.
